# threejs-shader-lab

A personal playground to experiment and learn custom shaders using **Three.js** and **GLSL**.

This project focuses on understanding how shaders work at a low level, including vertex transformations, fragment rendering, and dynamic visual effects.

---

## Live Demo

- [https://threejs-shader-lab.vercel.app/]

---

## 🚀 Features

- Custom **vertex and fragment shaders**
- Texture mapping with multiple selectable textures (GUI)
- Vertex displacement animations (wave effect)
- Real-time controls using **lil-gui**
- Built with **Vite** for fast development

---

## 🧠 What I'm learning

This project is part of my learning journey into shaders, one of the most challenging areas in 3D development.

Key concepts explored:

- How the GPU processes vertices and fragments
- Vertex displacement using mathematical functions
- Passing data via uniforms and attributes
- Working with textures inside shaders
- Understanding GLSL syntax and limitations
